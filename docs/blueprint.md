# **App Name**: FleetRush

## Core Features:

- Instant Quote & Booking: Users input delivery specifics like pickup/delivery locations, package details, and urgency to receive an immediate quote and finalize their booking.
- Real-Time Delivery Tracking: Provide a live interface for customers to monitor the status and geographical location of their active deliveries.
- Proof of Delivery Viewer: Enable secure access for customers to view and download digital proof of delivery for completed shipments.
- Informational Content Pages: Display comprehensive static content for Home, Services, Pricing, FAQ, and About sections, based on the provided document.
- Contact & Support Form: Implement an online form for users to easily submit inquiries or requests to FleetRush support, including name, email, phone, and message fields.

## Style Guidelines:

- Primary color: A sophisticated and energetic violet-blue (#4C26E6), symbolizing innovation and reliability. (HSL: 250, 70%, 50%)
- Background color: A subtle and clean very light purple-gray (#F6F5FB), ensuring content readability and a modern feel. (HSL: 250, 18%, 96%)
- Accent color: A vibrant electric blue (#5190FF), chosen to provide clear contrast for call-to-actions and key interactive elements. (HSL: 220, 90%, 65%)
- Headline font: 'Space Grotesk' (sans-serif), lending a modern, tech-forward, and bold impression.
- Body text font: 'Inter' (sans-serif), selected for its excellent readability and neutral, objective aesthetic, suitable for detailed descriptions.
- Utilize sleek, minimalist line icons to represent services and features, maintaining a professional and contemporary appearance.
- Adopt a clean, grid-based layout with ample whitespace to ensure easy navigation and focus on key information across all pages.
- Incorporate subtle hover effects on interactive elements and smooth transitions for section changes to enhance user experience without distraction.
